# How to Find Connected Accounts in Vercel

## Step-by-Step Instructions

### Step 1: Open Account Settings
1. Click your profile picture/icon (top right of Vercel dashboard)
2. Click **"Account Settings"** (it's in the dropdown menu you see)

### Step 2: Find Connected Accounts
Once you're in Account Settings, you'll see several tabs/sections:
- **Profile** (your name, email, etc.)
- **Security** (password, 2FA, etc.)
- **Connected Accounts** ← This is what you're looking for!
- **Billing** (if applicable)
- **Notifications**

Click on **"Connected Accounts"** tab.

### Step 3: Connect GitHub
1. In the Connected Accounts section, you should see:
   - GitHub
   - GitLab
   - Bitbucket
   - etc.

2. Find **GitHub** and click **"Connect"** or **"Connect GitHub"**

3. You'll be redirected to GitHub to authorize Vercel

4. After authorizing, come back to Vercel and you should see GitHub listed as "Connected"

## Alternative: Direct Link

You can also go directly to:
**https://vercel.com/account/connected-accounts**

## What You Should See

After connecting, you should see:
- ✅ GitHub (Connected)
- Your GitHub username (e.g., "BarclayM" or "dev-barometer")

This will fix the deployment access issue!
